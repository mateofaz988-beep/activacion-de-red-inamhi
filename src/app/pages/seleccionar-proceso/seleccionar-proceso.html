¿Cómo desea realizar su trámite?

Antes de iniciar, seleccione si el proceso se realizará de forma manual sobre papel o de forma electrónica mediante PDF firmado digitalmente.

Proceso físico

Proceso Manual

Registre sus datos básicos, obtenga un ID único, descargue el documento vacío en PDF y complete todo el trámite de forma física.

Nota importante

Si elige la opción manual, todo el proceso subsiguiente, incluyendo llenado, firmas y rúbricas, deberá realizarse estrictamente de forma física/manual sobre papel.

Indicaciones del proceso manual

  1. Llene un formulario corto con nombre, apellido y correo.
  2. El sistema generará un ID único de solicitud.
  3. Descargue el documento vacío en formato PDF.
  4. Imprima el documento y complételo manualmente.
  5. Firme y rubrique físicamente el documento.
  6. Escanee el documento finalizado.
  7. Ingrese a seguimiento con su ID y suba el PDF escaneado.
Proceso digital

Proceso Electrónico

Complete el formulario institucional, genere o descargue el PDF, fírmelo electrónicamente con FirmaEC y envíelo al flujo administrativo.

Uso de FirmaEC

FirmaEC es una aplicación de escritorio. El sistema web no firma directamente el documento; permite descargar el PDF y subir la versión firmada electrónicamente.

Indicaciones del proceso electrónico

  1. Llene el formulario institucional completo.
  2. Genere o descargue el documento PDF correspondiente.
  3. Abra el documento en FirmaEC desde su equipo.
  4. Firme el PDF con certificado digital o token físico.
  5. Suba el PDF firmado electrónicamente al sistema.
  6. El administrador continuará el flujo de aprobación.
  7. Al finalizar, recibirá la notificación institucional.

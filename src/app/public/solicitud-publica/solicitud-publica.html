Guía rápida

¿Cómo encontrar mi dirección IP?

La dirección IP identifica el equipo o red desde donde se solicita el acceso. Este dato ayuda al área técnica a revisar la solicitud correctamente.

¿Qué es una dirección IP?

Una dirección IP es un número que identifica un dispositivo dentro de una red. Un ejemplo común es 192.168.1.100. En este formulario se solicita para que TICS pueda validar desde qué equipo o red se requiere el acceso.

Opción 1

Ver IP usando comandos de Windows

  1. Presione las teclas Windows + R.
  2. Escriba cmd y presione Enter.
  3. En la ventana negra escriba ipconfig.
  4. Presione Enter.
  5. Busque el dato llamado Dirección IPv4.
  6. Copie ese número en el campo Dirección IP del formulario.
Opción 2

Ver IP desde Configuración

  1. Abra la Configuración de Windows.
  2. Ingrese a Red e Internet.
  3. Seleccione Wi-Fi o Ethernet, según su conexión.
  4. Abra las Propiedades de la red conectada.
  5. Busque el campo Dirección IPv4.
  6. Copie ese número en el formulario.
Importante: Si no conoce este dato, puede solicitar apoyo al área de TICS.
